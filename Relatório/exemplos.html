Exemplos "Hello World" em Frameworks Back-end

A seguir são apresentados exemplos básicos de implementação de uma rota que retorna a mensagem "Hello World" utilizando os frameworks citados no relatório.

Django (Python)


# views.py

from django.http import HttpResponse

def hello(request):
    return HttpResponse("Hello World")

Spring Boot (Java)


import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/")
    public String hello() {
        return "Hello World";
    }
}

Laravel (PHP)


<?php

use Illuminate\Support\Facades\Route;

Route::get('/', function () {
    return 'Hello World';
});

Express.js (JavaScript)


const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Hello World');
});

app.listen(3000);
